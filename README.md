# color_ex
A tool to collect information about colors and decisions


# to run server from root dir
* `DJANGO_ENV=development python backend/manage.py runserver_plus`

# to run frontend from `frontend` dir
* `yarn start`

# to reset the db
* `doit reset_db`

# other management things
* just use regular psql (have to install it on your machine..)
  * have to create a postgres user `color_ex` and grant it superuser perms
* just use regular django/yarn commands for backend/frontend respectively
